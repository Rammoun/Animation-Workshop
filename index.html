Les animations CSS

Définition:

Une animation CSS est une série de transformations qu'on attache à un élément HTML pour une durée choisie.

Quelques types de transformations

Transparence:

This is not transparent

This is partially transparent

<div class="transparent"></div> .transparent{opacity='0.5'}

Size:

This is scale=1

This is scale=1.5

<div class="scaled"></div> .scaled{transform: scale(1.5);}

Rotate:

This is rotated 60°

<div class="rotated"></div> .rotated{transform: rotate(60deg);}

Skew:

This is skewed

<div class="skewed"></div> .scaled{transform: skewX(20deg);}

Comment créer une animation CSS:

On crée une animation CSS en utilisant le mot clé @keyframes suivi par le nom de l'animation suivis par les frames. Example:

@keyframes fade{
from {opacity: 0;}
to {opacity: 1;}}

Comment attribuer une animation à un élément:

On a 2 méthodes:

La methode detaillee:

.slide-left{
animation-delay: 0s;
animation-direction: normal;
animation-iteration-count: 1;
animation-name: slide-left;
animation-timing-function: linear;
}

La methode reduite:

.fade{animation: fade(1) 1.5s(2) 2s(3) ease-in(4) 3(5);}

Explication de la methode reduite:

(1): anim.-name (2): anim.-duration; (3): anim.-delay (4):anim.-timing-function (5):anim.-iteration-count

I am created from HTML and fading in

.fade{animation: fade 1.5s 2s ease-in 3;}
@keyframes fade{
from {opacity: 0;}
to {opacity: 1;}}

I am created from HTML and rotating clockwise

.rotate-clockwise{animation: rotate-clockwise 4s 0s linear 1 ;}
@keyframes rotate-clockwise{
from{opacity: 0; transform: rotate(0)}
to {opacity: 1; transform: rotate(360deg)}}

I have the reverse of the rotate-clockwise animation

.rotate-clockwise-reverse{
animation: rotate-clockwise 4s 0s linear 1 reverse;}
@keyframes rotate-clockwise{
from{opacity: 0; transform: rotate(0)}
to {opacity: 1; transform: rotate(360deg)}}
